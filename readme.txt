WordPress WooCommerce Google Chrome Smart Lock facilitates the customer, store owner to login with saved password in their systems.

== Description ==

The admin can allow to save password to their nd login with that password. No need to remember password.

== Features ==
* The store owner can allow customers to login with their saved password.
* Also store owner can also select saved password to login.

== Installation ==

1. Install the WordPress WooCommerce Google Chrome Smart Lock
2. Activate the plugin
3. Go to the Chrome Smart Lock
